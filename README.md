# 🕌 Al-Haramain | Hajj & Umrah Spiritual Journeys

Al-Haramain is a premium, modern, and fully responsive spiritual travel agency portal. Designed to offer a premium experience for pilgrims planning their sacred journeys to Makkah, Madinah, and other Islamic heritage destinations worldwide.

---

## ✨ Features

- **Takeoff Loader Animation**: A unique, custom CSS/JS transition depicting a plane flight before revealing the website.
- **Glassmorphic Sticky Navbar**: A sleek navigation header that sticks to the top on scroll, transitioning smoothly into a legibility-enhancing blur & shadow effect.
- **Responsive Layouts**: Fully responsive pages tailored for all device widths (Desktop, Tablet, Mobile).
- **Featured Destinations & Packages**: Showcases detailed itineraries and features for Makkah, Madina, Turkey, Uzbekistan, and China.
- **Spiritual Mentorship Guides**: Integrated itineraries led by certified guides.
- **Direct WhatsApp & Booking Integration**: Sticky floating action buttons and booking prompts to instantly connect clients with travel experts.

## 🚀 Setup & Usage

Since this is a lightweight static website, you do not need to install any compilers or npm dependencies.

1. Clone this repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Double-click [index.html](file:///g:/hajj%20umrah%20web/index.html) to open the website directly in any web browser, or use a local development server extension (e.g. Live Server in VS Code).
